This is an attempt at remaking the software that Flock uses with their cameras. This Python program is very powerful and can track lots of faces and plates. It's not perfect yet, but overtime its does get better. I have added a function where the more you use it learns to detect faces and plates 

requirements:

pip install opencv-python opencv-contrib-python ultralytics paddleocr paddlepaddle flask yt-dlp

usage:

python flockv2.py

WARNING

I've only tested this on Debian, so idk if this works on Windows or Mac and shit, so if you're trying to use this on Windows, I recommend using WSL or something.
